New: added support for qualitative sizes as linewidth and markersize

linewidth can now be specified with 'xx-thin', 'x-thin', 'thin', 'medium', 'thick', 'x-thick', or 'xx-thick'. markersize can now be specified with 'xx-small', 'x-small', 'small', 'medium', 'large', 'x-large', or 'xx-large'. This change affects set_linewidth, and set_markersize; now allowing users to provide a string, or a float. A couple new rcParams were also added to support this change; lines.base.linewidth and lines.base.markersize which serve as the relative base value for the size constraints.

Below is an example of using the new feature in a LineCollection . 

import matplotlib.pyplot as plt
from matplotlib.collections import LineCollection
import numpy as np

N = 7
x= np.arange(N)
ys = [x + i for i in x]

fig,ax = plt.subplots()
ax.set_xlim(np.min(x), np.max(x))
ax.set_ylim(np.min(ys), np.max(ys))

line_segments = LineCollection([np.column_stack([x,y]) for y in ys], linewidths=('xx-thin', 'x-thin',
  'medium', 'thick', 'x-thick', 'xx-thick'))

line_segments.set_array(x)
ax.add_collection(line_segments)
plt.title('Collection of lines created with strings given as linewidth')

plt.show()

Below is an example of using the new feature with a couple large markers and a thin line. 

import matplotlib.pyplot as plt
import numpy as np

fig, ax = plt.subplots()

x = np.array([1,1])
y = np.array([0,9])

ax.plot(x, y, marker='o', markersize='xx-large', linewidth='thin')
ax.set_title('Large marker with a thin line')
